Package index

  • build.tagpage Generates page content for a tag, to be rendered with a layout

  • escape.html Escapes html tags

  • get.postpath returns the path to the post in the site
  • get.site.state Gets modification dates for all source and dest files in a site
  • html_body Wraps input in body tags
  • html_head Wraps input in head tags
  • html.postlist builds a list of links to all blog posts. List is in decreasing order of post date
  • html.taglist the absolute path to the Samatha site
  • img.link image tag representaion
  • import.tagfile Reads in a JSON tagfile
  • include.css CSS include representation
  • include.js js include representation
  • include.markdown a post file in markdown format
  • include.textfile NA
  • itemise.post a .md file of a blog post
  • javascript.tag NA
  • link.to link tag representaion
  • m Generic representaion of an HTML tag in R
  • mail.to mail to tag representaion
  • ordered.list ordered list representation
  • render.all.docs Builds a list of Samatha.Page objects for all Rd documents in a given directory
  • render.doc renders an .Rd file as a Samatha.Page object
  • render.opts Render html attributes
  • render.page Render a page using the Samatha html dsl
  • render.post Render a post from an R markdown file
  • render.rss the path to the site
  • render.tagfiles the absolute path to the Samatha site
  • rss.category NA
  • rssdoc Representation of an RSS page in R
  • samatha character absolute path to your Samatha site
  • skeleton path to the directory you want to set up
  • unordered.list unordered list representation
  • update.site Absolute path to your Samatha site
  • webdoc Representation of a webpage in R
  • write.docs Renders a directory of Rd files to html and writes within a Samatha site directory
  • write.html an object of class Samatha.Page
  • write.tags.to.file Path to the Samatha site
